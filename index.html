Skillsville Family Program

The Skillsville Family Program is a multimedia event series for families with children 5-8.

The program is designed to enhance children's executive function skills and knowledge about diverse career options.

To host your own Skillsville Family Program, review the materials below.

  • Pre-work
  • Kickoff
  • Focus
  • Organize
  • Feel
  • Remember
  • Think Differently

  • Pre-work

  • Confirm eligibility

    To use facilitate the Family Program you must meet the following requirements:

    • Ability to host 4 or more sessions.
    • Tablets for each child.
    • Reliable, stable internet, of at least 10 Mbps per user at the hosting location.
    • Ability to unblock/whitelist any websites needed for the program at the hosting location.
    • Confidence facilitating technology, and troubleshooting as needed.
    • Ability to print materials.
    • Ability to project materials.
  • Make your Skillsville Platform account

    The Skillsville Platform contains the episodes, Skillsville Strategies, digital games, and paper games for your family sessions. You will need to make an account and create childrenâ€™s accounts for the number of participants you expect to have.

    To create your account, go to bit.ly/Skillsville.

  • Choose your sessions

    The Skillsville Family Program is 6 sessions. Everyone must host the Kickoff Session first, then may choose to run the sessions in any order.

  • Kickoff

  • Review the Kickoff Session Slides

    Project these slides where the whole room can see at your Kickoff event.

    You can use the slide deck talking points as a guide.

  • Print Kickoff Session Materials

    Color printing is preferred, but all our materials will still work in black and white.

    • Volunteer Hand Out (1 per volunteer)
    • Skillsville Agenda (1 per family, and a few extras)
    • Kickoff Conversation Card (1 per family, and a few extras)
    • Skillsville To-Go (1 per family, and a few extras, print this double-sided)
    • Skillsville ID Badges (Made on the Skillsville Platform)
  • Play the Avatar Maker

    We recommend playing the game before your session so you can help others easily.

    Make an extra children's login to use. The Avatar Maker will pop-up when you login.

  • Complete the Kickoff Prep Checklist

    All items on the Kickoff Prep Checklist must be complete before the start of the Kickoff session.

  • After the Kickoff Session

    Now that everyone has made an Avatar in the Avatar Maker, reprint the Skillsville ID Badges.

    Staple each Skillsville ID Badge to a Resume Builder.

    You will hand out the Resumes at the start of each session going forward. Make sure each child has the Resume with their ID Badge on it.
